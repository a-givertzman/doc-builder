Осадки## Общие положения

Страница состоит из:

  • изображение судна с осадками, расположенное в верхней части страницы;
  • таблица с критериями посадки, расположенное в нижней части страницы.## Изображение судна На изображении судна отображаются осадки судна. В верхней части имеется поле выбора режима вывода осадок:
  • на перпендикулярах;
  • на марках;

  • наименование критерия с размерностью;
  • его расчетное значение;
  • условие выполнения (больше/меньше);
  • допустимое значение;
  • статус выполнения критерия для текущей загрузки.
