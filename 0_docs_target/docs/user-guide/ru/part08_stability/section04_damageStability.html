Аварийная остойчивость

Оценка аварийной остойчивости производится путем сравнения исправленной метацентрической высоты $h$ для текущего случая загрузки с минимально допускаемой метацентрической высотой деления на отсеки $h_$ для текущего водоизмещения (или осадки). Аварийная остойчивость для текущего случая загрузки считается удовлетворительной при выполнении условия $h \geq h_$. Результат расчета исправленной метацентрической высоты, минимально допускаемой метацентрической высотой деления на отсеки, а также статус выполнения критерия приводится в таблице критериев остойчивости.
